# Product Roadmap

## Phase 1 — Foundation (MVP)
- Recruiting Board
- Roster Builder
- Athlete Profile
- Basic Team Management Hub
- Initial training session logging
- Partial meet results ingestion
- Core schema + RLS patterns

## Phase 2 — Team Operations Expansion
- Full Practice Scheduler
- Workout Builder + Training Events
- Team Communications
- Role‑based dashboards
- Athlete training completion + readiness modeling

## Phase 3 — Meet Manager
- Meet creation + configuration
- Entries + seeding
- Live results + scoring
- Official results verification

## Phase 4 — AI Layer
- Scout Score
- Commit Probability
- Roster Projection
- Training Load / Readiness predictions

## Phase 5 — Multi‑Sport Expansion
- Universal Event Model implementation
- Sport modules: Swimming, Volleyball, Soccer, Basketball, etc.
- HS–College identity sync at scale.

## Phase 6 — Marketplace + Network Effects
- Athlete discovery network
- Verified results network
- Data export for NGBs and conferences
