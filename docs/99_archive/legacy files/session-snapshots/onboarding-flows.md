# Onboarding Flows – XC Ecosystem

## User Types
- **Coaches (Org-level)**
- **Athletes (Individual accounts)**

## Coach Flow
1. Create account  
2. Create program  
3. Invite staff  
4. Build rosters  
5. Begin using dashboards / recruit boards  

### Automation
- Auto-detect school via email domain  
- Auto‑populate program using universal-seed database  

## Athlete Flow
1. Create account  
2. Auto‑match to school  
3. Build profile (academic + athletic)  
4. Start training log, recruiting profile, accolades  

### Future: Guided onboarding wizard
