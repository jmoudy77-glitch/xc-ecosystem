# Entity Definition Law

Canonical entities are defined as:

- Program — a sovereign organizational unit operating under constitutional law  
- Team — a scoped competitive unit operating within a Program  
- Athlete — an individual subject to recruiting, performance, and compliance law  
- Season — a bounded execution period  
- Record — an immutable canonical determination produced by an execution engine  

Entities are constitutionally defined and may not be reinterpreted by UI, AI, or staff.
