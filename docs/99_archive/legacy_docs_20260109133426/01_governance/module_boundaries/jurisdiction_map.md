# Jurisdiction Map

| Module | Primary Jurisdiction |
|--------|----------------------|
| Program Health | capability, absence, readiness, structural constraints |
| Recruiting | pipeline, contact, evaluation, eligibility, roster inflow |
| Performance | training, load, execution, meet performance, readiness display |
| Compliance | rules, violations, enforcement, review, appeals intake |
| Billing | subscription state, entitlements, payment outcomes, access control |

Jurisdiction is exclusive unless explicitly delegated by amendment.
