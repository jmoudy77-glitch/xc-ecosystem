# Naming Conventions

## 1. Database Columns
- snake_case
- Meaningful names:
  - `athlete_id`
  - `team_season_id`
  - `created_by_user_id`

## 2. Enums
- Lowercase text enums (Supabase friendly).

## 3. Event Codes
- Use `SPORT_GENDER_EVENT`
- Examples:
  - `TFR_M_1500M`
  - `XC_W_5K`

## 4. Files
- Lowercase + hyphens:
  - `team-roster-table.tsx`
