# Roster & Scholarship Domain Model

## Purpose
Canonical meaning for roster composition, scholarship budgets, equivalencies/dollars, and audit history.

## Scope
Defines entities, relationships, and invariants. Does not define UI or implementation.

## Authoritative statements
- Define source-of-truth entities and lifecycle states for this domain.
- Define what can be derived vs what must be stored.
- Define boundaries with adjacent domains.
